## Require to run application

<p>PHP V8</p>
<p>Composer</p>


## Setting

1. Clone project
2. Create Database Mysql
3. overwrite DB setting in .ENV


## RUN APPLICATION

1. CLI (CMD) : "composer install"
2. CLI (CMD) : "npm install"
3. CLI (CMD) : "php artisan migrate:fresh --seed"
4. CLI (CMD) : "php artisan serve"
